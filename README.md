# TierlistTK
A tierlist GUI program in tkinter

# Features:-

- Supports saving
- Handles error when exception occurs
- File Detection with tierlist
- Beautiful yet minimal UI
- Keyboard shortcuts

# Keybindings:-

- Ctrl + O = Open a tierlist
- Ctrl + S = Save a tierlist
